示例4： string 字符串 数据类型

与java不同的是，javascript中没有字符的概念，只有字符串，所以单引号和双引号，都用来表示字符串。

示例5： var 动态类型 数据类型

变量的类型是动态的，当值是整数的时候，就是Number类型，当值是字符串的时候，就是String类型。

示例6：typeof 变量类型判断

使用typeof 来进行判断数据类型 正是因为 变量是动态类型，所以无法确定 当前到底是那种感觉数据类型，
这个时候，可以使用 typeof来进行判断
