js 中的对象是有着属性和方法的一种特殊数据类型

常见的对象有数字number，字符串string， 日期Date, 数组Array等

关键字
简介
new Number 创建一个数字对象
属性MIN_VALUE 最小值，最大值 属性MAX_VALUE
属性NaN 表示不是数字
方法 toFixed 返回一个数字的小数表达
方法 toExponential 返回一个数字的科学计数法表达
方法valueOf 答案-乘法
