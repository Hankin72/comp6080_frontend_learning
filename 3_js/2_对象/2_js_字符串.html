与数字类似，基本类型String也有一个对应的String 对象，并且提供了很多有用的方法。

new String() -------- 创建字符串对象 属性 length --------- 字符串长度 方法 charAt charCodeAt -------- 返回指定位置的字符 方法 concat --------- 字符串拼接 方法 indexOf lastIndexOf --------- 子字符串出现的位置 方法 localeCompare -------------- 比较两段字符串是否相同 方法 substring ----------------- 截取一段子字符串 方法 split -------------------- 根据分隔符，把字符串转换为数组 方法 replace ----------------- 替换子字符串
方法 charAt ----------------返回基本类型 方法 concat 方法 substring
